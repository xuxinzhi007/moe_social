# AutoGLM 系统架构

## 概述

AutoGLM是Moe Social项目中的智能助手系统，采用三层架构设计，包括前端任务规划器、Android原生无障碍服务和AI模型集成。本指南将详细介绍AutoGLM的系统架构和各组件之间的关系。

## 系统架构

### 三层架构设计

AutoGLM系统由以下三个主要层次组成：

1. **前端任务规划器**：负责用户交互、任务分解和执行调度
2. **Android原生无障碍服务**：处理设备操作、屏幕分析和事件监听
3. **AI模型集成**：提供意图理解、自然语言处理和决策支持

### 架构图

```
+------------------------+
|  前端任务规划器        |
|  (Flutter)            |
+------------------------+
|        ↑ ↓            |
|  方法通道/事件通道      |
|        ↑ ↓            |
+------------------------+
|  Android无障碍服务     |
|  (Kotlin)             |
+------------------------+
|        ↑ ↓            |
|  API调用/事件回调      |
|        ↑ ↓            |
+------------------------+
|  AI模型集成           |
|  (AutoGLM)            |
+------------------------+
```

## 核心组件

### 1. 前端任务规划器

前端任务规划器位于Flutter应用层，主要负责：

- **用户交互**：提供用户界面，接收用户指令
- **任务分解**：将复杂任务分解为可执行的子任务
- **执行调度**：协调各组件执行任务
- **状态管理**：跟踪任务执行状态

关键文件：
- `lib/autoglm/autoglm_page.dart`：AutoGLM主页面
- `lib/services/task_execution_engine.dart`：任务执行引擎
- `lib/services/memory_agent_service.dart`：内存代理服务

### 2. Android原生无障碍服务

Android原生无障碍服务负责与设备进行交互，主要功能包括：

- **屏幕分析**：分析当前屏幕内容
- **设备操作**：执行点击、滑动等操作
- **事件监听**：监听系统事件和用户操作
- **文本识别**：识别屏幕上的文本

关键文件：
- `android/app/src/main/kotlin/com/example/moe_social/AutoGLMAccessibilityService.kt`：AutoGLM无障碍服务
- `android/app/src/main/kotlin/com/example/moe_social/InputAssistAccessibilityService.kt`：输入辅助服务
- `android/app/src/main/kotlin/com/moe_social/moe_social/AutoGLMService.kt`：AutoGLM服务

### 3. AI模型集成

AI模型集成负责提供智能决策能力，主要功能包括：

- **意图理解**：理解用户的指令意图
- **自然语言处理**：处理和生成自然语言
- **决策支持**：为任务执行提供决策建议
- **记忆管理**：存储和检索用户记忆

关键文件：
- `lib/services/ai_inference_service.dart`：AI推理服务
- `lib/services/memory_service.dart`：记忆服务
- `lib/models/ai_memory.dart`：AI记忆模型

## 数据流

### 任务执行流程

1. **用户输入**：用户通过前端界面输入指令
2. **任务分解**：前端任务规划器将指令分解为子任务
3. **服务调用**：通过方法通道调用Android无障碍服务
4. **设备操作**：无障碍服务执行设备操作
5. **结果反馈**：将操作结果通过事件通道返回给前端
6. **AI分析**：AI模型分析结果并生成下一步建议
7. **任务完成**：前端更新UI，显示任务执行结果

### 数据传输

- **方法通道**：用于前端调用原生服务的方法
- **事件通道**：用于原生服务向前端发送事件
- **API调用**：用于与后端服务和AI模型通信

## 技术栈
