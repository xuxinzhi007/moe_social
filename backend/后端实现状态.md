## 📝 代码生成说明

### 关于 make proto 命令

**注意**：社区服务（community）相关的文件是**手动创建**的，不是通过 `make proto` 命令生成的。

以下文件是手动创建的，不会被代码生成工具覆盖：
- `api/internal/logic/community/*.go` - 11个文件
- `api/internal/handler/community/*.go` - 11个文件
- `model/group.go` - 群组模型

### make proto 的作用范围

`make proto` 命令主要用于：
- 生成 RPC 客户端代码
- 更新 protobuf 定义
- 生成 RPC 相关的存根

**不会影响**：
- 手动创建的 API handler 和 logic 文件
- API 层直接实现的业务逻辑
